1) CREATE :-


  •  Create Command is used to create a table in the database .

  • The CREATE TABLE command defines each column has minimum of three attributes .

  • Name
  • Data Type
  • Size (Column Width )

Syntax :- 

                            CREATE TABLE table_name (
    column_name1 data_type(size),
    column_name2 data_type(size);
);

Ex :- 

                            CREATE TABLE persons (
    Person_Id  int ,
    Last_Name varchar2(225) ,
    First_Name varchar2(225) ,
    Address varchar2(225) ,
    City varchar2(225) ,
);

2) ALTER


  • ALTER Command will be used to modify the structure of the existing table using ALTER we can do following things .
  • Add new columns to the table .

    Syntax :- 
                                    ALTER TABLE student ADD ( Location varchar2(30));
  • Removing existing column from table .

    Syntax :- 
                                    ALTER TABLE <table_name> DROP COLUMN column_name 
    ALTER TABLE student DROP COLUMN course
  • Modify columns data type size of the table

    Example :-   
    ALTER TABLE Student MODIFY (Name varchar2(20));
    • Change the name of the column of the table :-

    Syntax :- 
                                    ALTER TABLE <table_name> RENAME COLUMN <Old_name> TO <New_name>
    Example :- 
                                    ALTER TABLE student RENAME COLUMN Location TO Branch

3) TRUNCATE


  • TRUNCATE command used to remove the data from the table .
    Syntax :- 

    TRUNCATE TABLE <table_name>
    Ex :- 
                                      TRUNCATE TABLE Student

4) DROP


  • DROP Command used to remove table from the database .
  • It removes entire table from the database along with data present in it .

    Syntax :- 

    DROP TABLE <table_name>
    Ex :- 
                                      DROP TABLE Student

5) RENAME


  • RENAME Command used to change the name of database objects like table , view etc.
  • Using this command we can change only the name of database object not the columns of the table .

    Syntax :- 

     RENAME <Old_name> TO <New_name>
    Ex :- 
                                      RENAME Student TO Student_data

Practical No. 2

13/05/2021

Aim: - To study the DML Commands.
    Resources used Oracle 10g or SQL Server 2005 .

Theory: -

  • DML Commands are used to modify the data presented in the table.
  • These statement work on the data.
  • DML requires transaction control, any changes which is made to the database by DML statement will be called as transaction.
  • Any changes made by DML statement controlled by TCL statement .
  • DML Support following Categories of Commands ,
    1. INSERT
    2. DELETE
    3. UPDATE

1) INSERT: -

  • INSERT command will be used to insert new data into table .
  • In other words it will be used to create new rows in table .
  • While using this command the values are separated by commas & the values of data type varcahar , char , long are enclosed in single quotes.
  • We have 2 different syntax for writing Insert.
Syntax: - 

 INSERT INTO <table_name> VALUES (a_list_of_values)

Example: -

INSERT INTO dept VALUES (50,'IT','Pune');

   If data has to be written into only specific column this too is possible with slight variations to the general syntax.

Syntax:- 
INSERT INTO <table_name(column_name)> VALUES (a_list_of_values);
Syntax:- 

INSERT INTO emp(empno,ename,job,sal,hiredate) 
                  VALUES (101,'Vivek','Professor','40000',sysdate);

2) UPDATE: -

  • UPDATE command will be used to modify the data present in tables .
  • The update command consist of a ' set ' clause and an optional ' where ' clause to reflect changes in existing data UPDATE command is used .

Syntax: -

UPDATE <table_name> SET field1=value,field2=value WHERE condition;
Example: -

UPDATE dept SET dname='ACCOUNT' WHERE deptno=50 ;


Note:- If the " WHERE " condition is committed , the entire table is committed.


Eg.

 UPDATE emp SET job='DBA', sal=40000 WHERE empno=101;
Eg.

 UPDATE emp SET sal=(SELECT sal FROM emp WHERE empno=101) WHERE empno=7902;



3) DELETE: -

  • DELETE command will be used to remove data from table.
  • Using DELETE we can remove single row or rows from table.
  • The DELETE command consist the optional. ' where ' clause.
  • Two way we can delete the data.
Syntax: -

DELETE from <table_name> WHERE <condition>

First Way: -
Eg. 

DELETE FROM emp;

Second Way: - 

DELETE FROM emp WHERE job='DBA';
